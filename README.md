# DSR ENERGY x CURRENCY CASE STUDY

DS / ML final project. Emphasis on API maintenance and ETL, Exploratory Data Analysis and and Machine Learning Prediction.

QUESTION: To what extent is the Digital Silk Road going to drive trilateral energy equity (Electricity and Solar Energy) and consumer spending and currency stability in households by 2030?

Dataset: API driven Combination featuring reputable sources including official World Bank, DBNomics, etc.

Tech Stack: Pandas, Numpy, NashPy, SKlearn, Stats Model (data cleaning, intro / intermediate statistics + game theory, ML models)

Conjuction with a simpler, preceding EDA and Visualisation project, affecting the same trilateral trade relationship; culminating the two into a website via a JSON bridge
